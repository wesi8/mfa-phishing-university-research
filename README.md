# MFA vs Phishing in University Systems

## Abstract
This research evaluates the effectiveness of Multi-Factor Authentication (MFA) in mitigating phishing attacks within university systems. The study compares SMS OTP, TOTP, and hardware token authentication under simulated phishing conditions.

## Objectives
- Measure phishing success rate with and without MFA
- Compare MFA techniques
- Analyze user behavior
